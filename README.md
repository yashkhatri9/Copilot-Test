## ðŸš€ Quick Start

### Prerequisites

- Node.js (v18+ recommended)
- npm or yarn

### Installation

1. **Clone and setup backend:**
   ```bash
   cd backend
   npm install
   cp .env.example .env
   npm run dev
   ```
   Backend runs on **http://localhost:3000**
   Swagger docs at **http://localhost:3000/api-docs**

2. **Setup frontend (in a new terminal):**
   ```bash
   cd frontend
   npm install
   PORT=4200 npm start
   ```
   Frontend runs on **http://localhost:4200**
